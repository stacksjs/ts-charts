# API Overview

ts-charts consists of 30 packages, each corresponding to a D3.js module. Install them individually or use the umbrella `ts-charts` package.

| Package | Description |
|---------|-------------|
| [@ts-charts/array](/api/array) | Array manipulation, statistics, and data grouping |
| [@ts-charts/axis](/api/axis) | SVG axis generators for scales |
| [@ts-charts/brush](/api/brush) | Interactive brushing for selections |
| [@ts-charts/chord](/api/chord) | Chord diagram layout and ribbon generators |
| [@ts-charts/color](/api/color) | Color spaces (RGB, HSL, Lab, HCL, Cubehelix) |
| [@ts-charts/contour](/api/contour) | Contour and density estimation |
| [@ts-charts/delaunay](/api/delaunay) | Delaunay triangulation and Voronoi diagrams |
| [@ts-charts/dispatch](/api/dispatch) | Event dispatching |
| [@ts-charts/drag](/api/drag) | Drag interaction |
| [@ts-charts/dsv](/api/dsv) | CSV and TSV parsing/formatting |
| [@ts-charts/ease](/api/ease) | Easing functions for transitions |
| [@ts-charts/fetch](/api/fetch) | Data fetching utilities |
| [@ts-charts/force](/api/force) | Force-directed graph simulation |
| [@ts-charts/format](/api/format) | Number formatting |
| [@ts-charts/geo](/api/geo) | Geographic projections and path generators |
| [@ts-charts/hierarchy](/api/hierarchy) | Hierarchical data layouts (tree, treemap, pack) |
| [@ts-charts/interpolate](/api/interpolate) | Value interpolation (numbers, colors, strings) |
| [@ts-charts/path](/api/path) | SVG path serialization |
| [@ts-charts/polygon](/api/polygon) | Polygon geometry operations |
| [@ts-charts/quadtree](/api/quadtree) | Quadtree spatial indexing |
| [@ts-charts/random](/api/random) | Random number generators |
| [@ts-charts/scale](/api/scale) | Scales (linear, log, ordinal, time, etc.) |
| [@ts-charts/scale-chromatic](/api/scale-chromatic) | Color schemes and palettes |
| [@ts-charts/selection](/api/selection) | DOM selection and manipulation |
| [@ts-charts/shape](/api/shape) | Shape generators (line, area, arc, pie, etc.) |
| [@ts-charts/time](/api/time) | Time intervals and utilities |
| [@ts-charts/time-format](/api/time-format) | Time formatting and parsing |
| [@ts-charts/timer](/api/timer) | Timer and animation scheduling |
| [@ts-charts/transition](/api/transition) | Animated transitions |
| [@ts-charts/zoom](/api/zoom) | Pan and zoom interaction |
